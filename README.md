## 关于

Tiger AI Weekly 是一份专注于人工智能行业的专业周报，涵盖：

- 🤖 **大模型进展** - 最新模型发布、性能突破、开源动态
- 🚀 **AI应用落地** - 产品更新、企业采用案例、消费级应用
- 💰 **投资并购** - 融资轮次、并购交易、估值变化
- ⚖️ **政策法规** - 各国AI监管、政策变化、合规要求
